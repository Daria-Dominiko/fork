Git

  • git -v - выводит текущую версию Git.
  • git init - инициализирует локальный репозиторий
  • git clone *URL* - копирует удаленный репозиторий в текущую директорию
  • git status - отображает изменения в текущем репозитории
  • git add *имя файла* - добавляет файл в отслеживаемые
  • git commit -m "Имя коммита" - создание коммита
  • git push - отправляет изменения из локального репозитория на удаленный
  • git log - выводит список коммитов
  • git branch *имя ветки* - создает новую ветку
  • git checkout *название ветки* - переключается на ветку
