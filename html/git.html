Git

  • git init – Инициализирует новый Git-репозиторий в текущей директории.
  • git add – Добавляет изменения в индекс для последующего коммита.
  • git commit -m "сообщение" – Фиксирует изменения с сообщением.
  • git status – Показывает состояние рабочего каталога и индекса.
  • git push – Отправляет коммиты в удалённый репозиторий.
